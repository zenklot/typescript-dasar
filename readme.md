## How To Run Local

- Clone this repo
  ```sh
  git clone https://github.com/zenklot/typescript-dasar.git
  ```
- Install dependencies
  ```sh
  yarn install
  ```
- Run Watcher
  ```sh
  yarn ts
  ```
- Run Development
  ```sh
  yarn dev
  ```
